Reports

PDF and Excel reports are available on demand. Please contact Manus to request report generation for your projects.

The following report types can be generated for you:

  • Total Project - Hours (PDF & Excel)
  • Total Project - Quantities (PDF & Excel)
  • Sub Job - Hours (PDF & Excel)
  • Sub Job - Quantities (PDF & Excel)

When requesting a report, please specify:

  • The Project Name (and Project ID if known)
  • The Report Type (e.g., Total Project - Hours)
  • For Sub Job reports, the specific Sub Job Name (and Sub Job ID if known)

The direct in-application export functionality is currently unavailable due to deployment environment limitations with certain specialized libraries. We appreciate your understanding.
